## Contribution Templates

### Template 1: Documentation PR
```markdown
## Add [Feature] Example

### Changes
- Added example for [feature]
- Includes code snippet
- Explains common use case

### Why
- Helps new developers get started
- Addresses common question in Discord
- Reduces support burden

### Testing
- Verified code compiles
- Tested on Base Sepolia
- Checked all links work
```

### Template 2: Bug Fix PR
```markdown
## Fix: [Issue description]

### Problem
- Describe the bug
- How to reproduce

### Solution
- What you changed
- Why this fixes it

### Testing
- How you tested
- Results

Fixes #[issue number]
```

### Template 3: Issue Template
```markdown
## [Clear, specific title]

### Description
Clear explanation of issue/feature

### Expected Behavior
What should happen

### Actual Behavior
What actually happens

### Steps to Reproduce
1. Step one
2. Step two
3. Step three

### Suggested Solution
(Optional but helpful)

### Environment
- Base Mainnet/Sepolia
- Browser/Node version
- Relevant versions
```
